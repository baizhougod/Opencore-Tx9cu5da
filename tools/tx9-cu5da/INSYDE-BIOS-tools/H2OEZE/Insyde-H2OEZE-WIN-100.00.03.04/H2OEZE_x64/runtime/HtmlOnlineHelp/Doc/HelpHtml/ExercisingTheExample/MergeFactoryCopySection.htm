Merge Factory Copy Section

To merge a factory copy file into the current BIOS image having factory copy section, you can follow these steps:
1. Select the Factory copy merge menu item from the Other menu
 2. Press Browse button and a open file dialog is showen, and select a factory copy file
3. The tool will check if the BIOS image has factory copy section. If yes, the size of the factory copy section will be compared with the input file size. If the results are positive, the merge operation will be processed.
