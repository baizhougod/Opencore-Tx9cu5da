Replace Firmwre Volumn

To replace a Firmware Volumn in FD file, follow these steps:

1. Read the FD file (refer to Modify Logo step 1).

2. Select Replace FV in the FV option from the Components menu. The Replace FV window appears.

3. Indicate the FV file path and specify a FV you would like to supersede.


Replace FV

4. Click the Apply button to start processing. The tool will show an error message dialog if failed, or display a "Success" message in status bar if successful.
