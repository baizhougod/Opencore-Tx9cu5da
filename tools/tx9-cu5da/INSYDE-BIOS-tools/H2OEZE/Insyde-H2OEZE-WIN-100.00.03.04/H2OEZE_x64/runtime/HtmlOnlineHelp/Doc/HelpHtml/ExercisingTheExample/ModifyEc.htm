Modify EC

To modify EC, follow these steps:

1.  Read the FD file (refer to Modify Logo step 1). If EC exists in the file, the Binary-Rom function on the menu bar is enabled, allowing you to select the EC option.

2. When you select the EC option, the Ec Rom window appears.

Ec Rom window

3. Read the binary file of Ec Rom.

4. Push the Apply button to begin modifying EC. Once the modification has been successfully completed, the Ec Rom window is closed, and the lower-left turn corner of the main screen displays the message "FD file modify finished."
