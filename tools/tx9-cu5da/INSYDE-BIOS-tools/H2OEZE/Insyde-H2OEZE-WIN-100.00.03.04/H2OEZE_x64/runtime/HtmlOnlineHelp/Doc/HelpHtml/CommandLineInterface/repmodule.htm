repmodule

        -repmodule:[ImagePathName] -modulepath:[ModulePathName] -fvnum:[FvNum] -out:[OutputPath]

 

To replace a module, the following commands are also required:

   -modulepath: specify the module file path

   -fvnum: specify the number of the target Fv (zero-based)

 

The following command is optional:

   -out: specify the bios image's output file path. If the output file path is not specified, the operation result will be saved to the original one (i.e., [ImagePathName]).
