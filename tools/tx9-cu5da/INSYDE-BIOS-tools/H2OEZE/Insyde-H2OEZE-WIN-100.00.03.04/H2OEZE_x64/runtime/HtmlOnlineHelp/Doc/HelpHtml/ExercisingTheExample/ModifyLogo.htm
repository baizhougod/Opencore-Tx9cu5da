Modify Logo

To modify a logo, follow these steps:

1. Open the FD file to be modified:

  • Select the Load File option via the File function


Selecting the Load File option
  • Specify the FD file


Specifying the FD file
  •  Show the FD information


Showing the FD information
  • Finish loading the FD file.

 

Note:

The FD file is a pure bios image which only includes the images or binary data defined in flashmap, not including ME data.

 

2. Display the Logo Image window:

  • Select the Logo option via menu Logo from the Components menu, or expand Conponents node and then click Logo subitem in the tree on the left window.


Selecting the Logo option

3. Specify the format of image file. You can also press the View button to preview the image file.


Specifying the logo and viewing the image file

4. Specify the logo image to be loaded
Specifying the logo image

5. Press the Apply button to replace the image file specified. If both files are not consistent in format or size, a window shows asking you to convert. Press yes to convert, otherwise preserve present file format to substitute for.
Specifying the logo image and conversion

6. Upon completion of modification, the lower-left turn corner of the main screen displays the message "Success"
