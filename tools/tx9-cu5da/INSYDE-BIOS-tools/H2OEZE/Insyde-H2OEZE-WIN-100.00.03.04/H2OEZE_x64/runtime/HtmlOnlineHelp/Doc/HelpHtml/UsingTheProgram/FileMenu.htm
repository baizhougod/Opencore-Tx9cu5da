File Menu

To use the File function, select File from the menu bar. A drop-down menu appears as the following.


 

Load BIOS image from

Read any FD file. Upon loading the file, you can then select other options in this menu like Close, Save or Save As; or use such menu-bar functions as BIOS image, Components, Binary-Rom, Other, ROM-Translation, etc.

Note:

The FD file is a pure bios image which only includes the images or binary data defined in flashmap, not including ME data.

Close BIOS image

Close the file in use.

Log

Create the H2OEZE.ini to record action that you use the utility with H2OEZE.

Save

Save the file back to the FD file.

Save As

Save as a new file.

Exit

Exit the current program.
