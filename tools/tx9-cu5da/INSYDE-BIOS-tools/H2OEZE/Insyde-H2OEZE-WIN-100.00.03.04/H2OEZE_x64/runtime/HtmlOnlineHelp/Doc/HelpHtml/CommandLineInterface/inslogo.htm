inslogo

        -inslogo:[ImageFilePath] -logopath:[LogoImageFilePath] -guid:[Guid] -fvnum:[FvNum] -out:[OutputPath]

 

To generate a logo module, the following commands are also required:

   -logopath: specify the logo file path

   -guid: specify the logo module!|s GUID

 -fvnum: specify the number of the target Fv (zero-based) to insert

 

The following command is optional:

   -out: specify the output file path. If the output file path is not specified, the operation result will be saved to the original one (i.e., [ImageFilePath]).
