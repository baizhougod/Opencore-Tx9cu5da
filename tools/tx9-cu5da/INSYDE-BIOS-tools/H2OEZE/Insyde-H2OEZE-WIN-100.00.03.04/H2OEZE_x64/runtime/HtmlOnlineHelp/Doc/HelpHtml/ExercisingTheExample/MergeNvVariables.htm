Merge Nv Variables Data

In this processing, the tool will retrieve the data in the variable store region and factory copy region from an input file and overwrite to the BIOS image. To do this, following these steps:
1.      Select the NV Variables Merge menu item from the Other menu
2.      A Open File dialog is showed, and select a binary file which has NV Fv data
3.      The tool will check the used variable data size in the variable store region and check the factory copy region size. If the checks pass, the merge operation will be processed.
