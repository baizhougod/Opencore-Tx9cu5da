replogo

        -replogo:[ImagePathName] -logopath:[LogoImagePathName] -logoindex:[Index id] -logoconvert:[0/1] -out:[OutputPath]

 

To replace a logo, the following commands are also required:

  -logopath: specify the logo file path

  -logoindex: specify the logo module's index id

  -logoconvert: specify whether to convert input image format to FD image format(1), or to keep input image format(0)

  

Use the following command to dispaly available logos:

        -replogo:[ImagePathName]

 

The following command is optional:

   -out: specify the bios image's output file path. If the output file path is not specified, the operation result will be saved to the original one (i.e., [ImagePathName]).
