repoprom

        -repoprom:[ImagePathName] -oprompath:[OpRomPathName] -opromindex:[Index id] -out:[OutputPath]

 

To replace the option rom, the following commands are also required:

   -oprompath: specify the option rom file path

   -opromindex: specify the module's index id

 

Use the following command to display the available option roms:

        -repoprom:[ImagePathName]

 

The following command is optional:

   -out: specify the bios image's output file path. If the output file path is not specified, the operation result will be saved to the original one (i.e., [ImagePathName]).
