Replace Module

To replace a module, follow these steps:

1. Read the FD file (refer to Modify Logo step 1).

2. Select Replace module in the Module option from the Components menu. The Replace module window appears.

3. Specify the module file path and the corresponding GUID.


Replace module

4. Click the Apply button to start processing. The tool will show an error message dialog if failed, or display a "Success" message in status bar if successful.
