BIOS Image Menu

To use the functions, select BIOS image from the menu bar upon loading the file. A drop-down menu appears for you to select the option.

 

Compare with 2nd BIOS Image

Select another image file and click “Apply” button, and then will show the different between them.


Select 2nd BIOS image


The result of comparison appears three distinct color status. Black represents totally alike content, red indicates some different in content, and purple illustrates the item only existed one file.


Different between two image files

Three view mode can be shown, as following:

  • All        –  Show all items including the same and non-equivalent content
  • Diffs     –  Show only different items
  • Same    –  Show equivalent items


Three view mode

It will shows a new page that appears raw data comparing with another one when double clicking one item.
