Modify Default Settings of Setup Utility

To modify default settings of the Setup Utility, follow these steps:

1.      Read the FD file (refer to Modify Logo step 1).

2.      Select the Setup Utility option via the other function. The Setup Utility window appears, showing the initial settings of BIOS.


Setup Utility window

3.      Use the mouse to modify the setting as required.


Setup Utility modify window

4. To exit, press the Exit memu list, click Save options to store the changes, or Discard options with nothing saved.


Setup Utility exit window
