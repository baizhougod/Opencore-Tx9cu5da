Other

To use the other function, select other from the menu bar upon loading the file. A drop-down menu appears for you to select the option.

 

Microcode

Selecting Microcode shows a window. Through this Microcode window, you can change the microcode.

 
Microcode window

 Name
  Description
 Delete
  Delete the selected microcode
 Add
 Select a bin file to append the input microcode at the end
 Relpace
  Select a bin file to replace the selected microcode
 Relpace all
  Select a bin file to replace whole microcode
 Extract
  Extract the selected microcode to a bin file
 Extract all
  Extract whole microcode to a bin file


Setup Menu - Update default

Selecting the Setup Utility option displays a window. Through this window, you can read the BIOS setup information and modify the initial value of basic option. To store this information, you may press Patch from the Exit drop-down menu.


Setup utility window

Setup Menu - Update string

To update SCU strings, select Other->Setup menu->Update string menu item. To modify in another language, select the language in the Lang combo box. Once finishing editing, click "Ok" button to save back, or "Cancel" button and discard the changes before modified.

Factory copy marge

  • Enabled if the BIOS image has factory copy section
  • To merge a factory copy file into the current BIOS image having factory copy section, follow these steps:
  1. Select the Factory Copy Merge menu item from the Other menu
  2. An Open File dialog is showed, and select a factory copy file
  3. The size of the factory copy section will be compared with the input file size. If the results are positive, the merge operation will be processed
  4. The status bar will  display message to notify completion of the operation
