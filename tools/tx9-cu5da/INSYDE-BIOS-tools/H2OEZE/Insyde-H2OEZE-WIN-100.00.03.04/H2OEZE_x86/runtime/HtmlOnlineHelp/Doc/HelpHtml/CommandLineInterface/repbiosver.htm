repbiosver

        -repbiosver:[ImagePathName] -biosver:[biosVersion] -out:[OutputPath]

 

To replace the bios version, the following commands are also required:

   -biosver: specify the new bios version

  

Use the following command to display the current bios version:

        -repbiosver:[ImagePathName]

 

The following command is optional:

   -out: specify the bios image's output file path. If the output file path is not specified, the operation result will be saved to the original one (i.e., [ImagePathName]).
