Insert Module

To insert a non-existing module, follow the steps:

1.      Read the FD file (refer to Modify Logo step 1).

2.      Select Insert Module in the Module option from the Components menu. The Insert module window shows on the rigth side.


Insert New Module

3.      Specify the module file path and choose the correct FV listed in the Select Insert FV box.

4.      Click the PatchPatchPatchPatch button to start processing. The tool will show an error message dialog if failed, or display a "Success" message in status bar if successful.
