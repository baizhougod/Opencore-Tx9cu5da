Generate Logo FFS

To generate logo FFS, follow these steps:

1. Select the Generate Logo FFS option from the Other menu.

2. Select FFS type. The utility support EDK and EDK2 formats.

3. Specify the logo file. The utility support JPG, BMP, TGA, GIF and PCX formats.

4. Specify the path of FFS file where do you want save it.

5. Make sure all the settings are correct. Press the Apply button to generate the FFS file.
