Replace Option Rom

To replace the option rom, follow the steps:

1.  Read the FD file (refer to Modify Logo step 1).

2. Select OpROM from the Binary-ROM menu. The Replace OpROM window appears.


OpROM window

3. Specify the option rom file path and the corresponding GUID.

4. Click the Apply button to start processing. The tool will show an error message dialog if failed, or display a "Success" message in status bar if successful.
