Components Menu

To use the Components function, select Components from the menu bar upon loading the file. A drop-down menu appears for you to select the option.

 

Logo

Through this Logo Image window, you can change the logo in your FD file; the PCX, JPG, TGA, GIF and BMP formats are supported.


Logo Image window
  • Logo Image from: Specify the new graphic file.
  • Current internal logo: Upon reading the FD file, all the GUIDs of the graphic files in it are converted to the image information, e.g., format (pixel * pixel).
  • View: Use this button to preview the picture selected.

If the format of the picture isn't consistent with that of the FD image, a message appears, asking if you want to convert the FD image format into that of the picture. Give your choice as needed.


Convert message

Replace module

Selecting the Module Replace option show a window. Through this Module Replace window, you can change the module; the PEI, ORG, and DXE formats are supported.


Module replace window
  • Load module from: Specify the file path of the new module.

Insert module

Selecting the Insert New Rom option on the Components menu shows a window. Through this Insert New Module window, you can insert the module; the ORG, FFS, PEI and DXE formats are supported.


Insert module window
  • Load module from: Specify the new module.
  • Select FV to insert: Select the FV to insert.

Delete module

Selecting the Delete Existing Module option shows a window. Through this delete module window, you can delete the module.


Delete module window
  • Select moudle to delete: Specify the GUID for the module to be deleted.

Export Module

To export a module as a file, select the module by guid and specify the output file path, and press the "Apply" button to export.


Export module window

Replace FV

Selecting the Replace FV option shows a window. Through this Replace FV window, you can replace the FV.


Replace FV window
  • Load FV from: Specify the new FV.
  • FV: Select the FV to be replaced.

NOTE:

The user has to check that the size of the input file is the same with the one packed in bios image. Currently, it is only allowed to replace FvMain (in which DXE modules are saved) and FvRecoverty (in which PEI modules are saved).
