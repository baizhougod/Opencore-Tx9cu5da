Export Module

To export a existing module, follow the steps:

1. Read the FD file (refer to Modify Logo step 1).

2.  Select Export module in the Module option from the Components menu. The Export module window appears.


Export module

3. Select the GUID of the module to be exported in the Module Information list box.

4. Indicate a file name and press the Apply button to start processing. The tool will show an error message dialog if failed, or display a "Success" message in status bar if successful.
