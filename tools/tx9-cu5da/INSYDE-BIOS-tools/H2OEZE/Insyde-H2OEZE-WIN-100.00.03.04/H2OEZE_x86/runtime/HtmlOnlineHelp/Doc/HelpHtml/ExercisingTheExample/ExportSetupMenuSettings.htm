Export Setup Menu Settings

To export setup menu settings, follow the steps:

1. Read the FD file (refer to Modify Logo step 1).

2.  Select Export setup menu settings in the Setup menu option from the Other menu. The Export setup menu settings window appears.


Export setup menu settings

3. Indicate a file name and press the Apply button to start processing. The tool will show an error message dialog if failed, or display a "Success" message in status bar if successful.
