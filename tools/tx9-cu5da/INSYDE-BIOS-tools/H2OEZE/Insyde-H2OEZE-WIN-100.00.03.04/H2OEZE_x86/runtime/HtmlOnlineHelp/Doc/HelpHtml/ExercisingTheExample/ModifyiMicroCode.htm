Modify Microcode

To modify the microcode, follow these steps:

1. Read the FD file (refer to Modify Logo step 1).

2. Select the Microcode item in the Other menu. The Microcode window will be showen.

3.  Six functions are provided:
  • Delete - delete the microcode data which is selected in the Microcode detail box
  • Add - append the input microcode file data at the end
  • Replace - replace the whole microcode data with the input microcode file
  • Replace all - Select a bin file to replace whole microcode
  • Extract - Extract the selected microcode to a bin file
  • Extract all - Extract whole microcode to a bin file


Microcode window

4. Once the modification has been successfully completed, the lower-left turn corner of the main screen displays the message "Success"
